# xuanyu ui uniapp

基于 Vant 轻量、可靠的 UniApp 组件库

## 开发者信息
- 作者：孟祥涵
- QQ：1056811341
- 微信：meng_xianghan

## 使用

1. 将项目中 xuanyu-ui 目录复制到你的项目中
2. 在 App.vue 文件中引用 xuanyu-ui/index.scss

注：demo在项目中 element 文件加中，将项目拉取到本地预览即可
